# Путь в системные аналитики 📊

Открытый конспект, который я веду по ходу освоения профессии **системного аналитика** с нуля (с упором на ИИ как рабочий инструмент).

**🔗 Читать сайтом:** https://dmtrvrnv.github.io/systems-analyst-handbook/

Это не платный курс и не сертификат — это живая рабочая тетрадь: конспекты с разбором на реальных примерах. Все термины расшифрованы прямо в тексте, гуглить не нужно. Если ты тоже входишь в профессию с нуля — забирай, вдруг пригодится.

## Что уже есть

| Модуль | Тема | Статус |
|---|---|---|
| 0 | Кто такой системный аналитик | ✅ |
| 1 | Требования (актор/стейкхолдеры, проверяемость, функц/нефункц) | ✅ |
| 1+ | Критерии приёмки | ⏳ |
| 2 | Нотации: BPMN и UML | ⏳ |

Дальше по плану: статистика, API и интеграции, архитектура систем, документация и ТЗ, портфель.

## Как читать

Открой `index.html` в браузере или заходи на сайт по ссылке выше. Слева — меню по модулям, файлы пронумерованы, читаются по порядку.

## Как это сделано

Чистый HTML + CSS, без фреймворков. Схемы нарисованы через SVG (векторные, подстраиваются под тёмную/светлую тему). Хостинг — GitHub Pages.
